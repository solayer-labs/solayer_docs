# Solayer Card Fee Structure

Solayer offers a transparent and user-friendly fee structure designed to provide a seamless and cost-effective financial experience. Below are the key details regarding fees associated with using the Solayer Card.

## Card Maintenance & Inactivity Policy

To maintain an active user base, the following inactivity policy applies:

- Cards will be **deactivated after 6 months** of inactivity.
- To continue using the card, users must request a **new card issuance**.

<br/>

## Top-Up & Transaction Fees

Solayer ensures transparent and minimal fees for top-ups and transactions:

| Fee Type                | Amount                                  |
|-------------------------|------------------------------------------|
| Top-Up Fee              | 1% per top-up transaction                |
| USD Transaction Fee     | $0.15 per transaction                    |
| International Fee       | 0.10% + 1.5% per international transaction |
