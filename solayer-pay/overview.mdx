## Powered by Solayer Chain: Real-Time On-Chain Processing

Solayer Pay is built on Solayer Chain, a hardware-accelerated Solana Virtual Machine (SVM) that delivers high throughput with real on-chain execution.

- Hardware-Accelerated: Built with RDMA & InfiniBand for ultra-low latency
- On-chain settlement that rivals centralized processors
- DeFi-Ready: Supports SVM and EVM chains for cross-chain compatibility (Coming Soon)

With Solayer Chain, every Solayer Pay transaction settles on-chain with low latency.
