# 

## Prerequisites

### Install Rust \+ Cargo

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source ~/.cargo/env
```

### Install Solana CLI v1.18.0\+

```bash
sh -c "$(curl -sSfL https://release.solana.com/v1.18.0/install)"
export PATH="~/.local/share/solana/install/active_release/bin:$PATH"
```

### Install BPF Toolchain

```bash
rustup component add rust-src
```

---

## Step 1: Create Project Structure

```bash
mkdir solayer-hello
cd solayer-hello
cargo new --lib hello
```

## Step 2: Configure Cargo.toml

Edit `hello/Cargo.toml`:

```toml
[package]
name = "hello"
version = "0.1.0"
edition = "2021"

[dependencies]
solana-program = "1.18.0"

[lib]
crate-type = ["cdylib", "lib"]
```

## Step 3: Write the Hello World Program

Replace `hello/src/lib.rs` with:

```rust
use solana_program::{
    account_info::AccountInfo, entrypoint, entrypoint::ProgramResult,
    pubkey::Pubkey, msg,
};

entrypoint!(process_instruction);

fn process_instruction(
    _program_id: &Pubkey,
    _accounts: &[AccountInfo],
    _instruction_data: &[u8],
) -> ProgramResult {
    msg!("Hello, Solayer!");
    Ok(())
}
```

## Step 4: Build the Program

```bash
cd hello
cargo build-sbf --sbf-out-dir=../dist
cd ..
```

> **Note**: Use `cargo build-sbf` (not `cargo build-bpf` as in older guides)

This creates the binary at `dist/hello.so`

## Step 5: Generate Keypairs

```bash
# Generate program keypair
solana-keygen new -o hello_program-keypair.json --no-bip39-passphrase

# Generate wallet keypair
solana-keygen new -o your-wallet-keypair.json --no-bip39-passphrase

# Get your program ID
solana address -k hello_program-keypair.json
```

**Save the Program ID** - you'll need it for deployment verification.

## Step 6: Configure Solayer Devnet

```bash
solana config set --url https://devnet-rpc.solayer.org
solana config set -k ./your-wallet-keypair.json
```

## Step 7: Fund Your Wallet

```bash
# Request test SOL (may need multiple attempts)
solana airdrop 2
solana airdrop 1
solana balance  # Should show ~0.3 SOL
```

> **Note**: You need at least 0.15 SOL for deployment

## Step 8: Deploy the Program

```bash
solana program deploy \
  -u https://devnet-rpc.solayer.org \
  --use-rpc \
  --program-id ./hello_program-keypair.json \
  --upgrade-authority ./your-wallet-keypair.json \
  ./dist/hello.so \
  -k ./your-wallet-keypair.json
```

> **Note**: This deploys a new program. To upgrade an existing program, use the same command with the existing program's keypair.

## Step 9: Verify Deployment

```bash
solana program show <YOUR_PROGRAM_ID> -u https://devnet-rpc.solayer.org
```

> **Note**: May show formatting error - this is safe to ignore

## Troubleshooting

| Issue                                | Solution                                                  |
| ------------------------------------ | --------------------------------------------------------- |
| **Build fails with lock file error** | Delete `Cargo.lock` and rebuild                           |
| **Insufficient funds error**         | Request more SOL: `solana airdrop 1`                      |
| **RPC connection issues**            | Ensure using `--use-rpc` flag                             |
| **Program show formatting error**    | Safe to ignore - deployment was successful                |
| **Transfer to program fails**        | Expected behavior - programs cannot receive SOL transfers |
